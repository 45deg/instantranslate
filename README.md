# Instantranslate

Translating the text of the clipboard.

![demo](https://raw.githubusercontent.com/45deg/instantranslate/demo/demo.gif)

## How to install

Requirements: node, yarn

```
yarn # Installation
yarn dev # start
```

or see [releases](https://github.com/45deg/instantranslate/releases).
